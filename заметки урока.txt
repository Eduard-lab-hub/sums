Тема: Git и Github
Github - система управления версиями проектов(общих или личных)

1) Регистрация на Github ---> https://github.com/
2) Скачать Git
3) Окрыть терминал и проверить версию Git
            git --version

Тема: Создание репозитория git, выгрузка проекта на git, Commits git, удаление репозитория
1) Открыть терминал.
2) Прописать следующие команды:
            I -  команда: git init             # инициализация вашего проекта
            II - команда: git add .            # сохранение 
            III- команда: git commit -m "Ваш коментарий к репозиторию"     # коментарий, что именно было сделано
